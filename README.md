### Description
구글, 티스토리, 깃헙 등등 프론트엔드 면접에서 자주 물어보는 질문 자료들을 정리하여 만든 사이트입니다.  
PWA(Progressive Web App)를 적용하여 반응형 모바일 웹앱으로 개발을 하였고, 모바일에서 앱을 설치하여 사용할 수 있습니다.

### Package
- Next.js 
- React 18
- Emotion
- SCSS
